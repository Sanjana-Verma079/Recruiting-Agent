# Recruiting-Agent
An intelligent recruitment automation system built using LangGraph, LangChain, and LLMs. The application analyzes candidate applications, classifies experience levels, evaluates skill compatibility for a Python Developer role, and automatically routes candidates through the appropriate hiring workflow.


## Features

* Candidate experience classification
* Skill assessment for Python Developer roles
* Automated recruitment workflow
* HR interview scheduling
* Recruiter escalation for senior candidates
* Candidate rejection handling
* LangGraph workflow visualization

## Tech Stack

* Python
* LangGraph
* LangChain
* OpenAI
* Google Colab

## Workflow

Application → Experience Classification → Skill Assessment → HR Interview / Recruiter Escalation / Rejection

## How to Run

1. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

2. Configure your OpenAI API key.

3. Run the notebook:

   ```bash
   jupyter notebook
   ```
